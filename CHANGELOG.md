# Changelog

项目版本变更记录。

## [v1.2.1] - 2026-06-15

### Added

- 新增版本发布说明文档 [docs/release_notes_v1.2.1.md](docs/release_notes_v1.2.1.md)。
- 为 AI 重点动作新增分组元数据、动作概览卡和统一动作网格渲染能力。
- 为报告渲染新增重点字段加粗高亮规则，并补充对应测试用例。

### Changed

- 固定月报输出文件名与报告标题为 `YYYYMM-月度纸袋分析报告`，不再附带运行时随机后缀。
- 将“本月优先执行事项”重构为正式版管理视图：
  - 顶部保留 `库销与库存类 / 纸袋使用合规类 / 终端异常 / 盘点类` 三个摘要卡。
  - 下方动作卡改为统一 3 列等高网格，每张卡片自带类别标签和优先级标签。
- 将“AI 重点行动清单”改为按 `P1 / P2` 和动作类别分组展示，并增加动作总览卡。
- 优化“核心结论”“经营摘要”“黄灯大区简要提示”等模块的阅读层级，使核心问题、管理动作、关键数值更容易定位。

### Fixed

- 修复报告中多处重复问题标签，例如 `库存结构不科学 / 库存结构不科学`。
- 修复黄灯大区简要提示中库存问题与建议文案重复堆叠的问题。
- 修复“地区焦点 / 专项焦点 / 本月优先执行事项”排版不一致、卡片长短不齐的问题。
- 修复“核心结论”与“经营摘要”之间的信息重复，改为前者偏管理判断、后者偏快速导读。

### DingTalk Delivery

- 收口钉钉正式发送链路，当前主路径为企业内部应用 + `convFile` 群文件发送。
- 保留 H5 JSAPI 绑定页与本地签名服务，用于一次性采集并保存 `openConversationId / unionId / userId / operatorName`。
- 支持按月直接重发已生成 PDF 文件，并复用既有群绑定。
